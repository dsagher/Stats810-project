
# Smoking Health Data Analysis

## Dependencies (Python)
```
pandas
matplotlib.pyplot
numpy
seaborn
sklearn
```

## Files

Both files below can be run in any order

- main.py - contains code for Principal Component Analysis and Logistic Regression. Written by Ivan Bega and Dan Sagher
- EDA.py - containds EDA part of the project. Written by Ethan Richards


## Dataset
The "Body signal of smoking" dataset was used. It is a synthetic dataset with 55692 records and 27 features. It can be accessed at the link: https://www.kaggle.com/datasets/kukuroo3/body-signal-of-smoking
